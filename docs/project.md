# 프로젝트 개요

## 이름
AI Academy Advisor

## 목표
하남 미사 지역의 수학학원을 AI가 추천해주는 도메인 특화 AI 서비스의 MVP를 개발한다.

## 배경
학부모/학생이 지역 내 다수의 수학학원 중 자신에게 맞는 학원을 찾기 어렵다는 문제를 해결하기 위해,
학원 정보(커리큘럼, 대상 학년, 위치, 후기 등)를 구조화하고 AI가 사용자 조건에 맞춰 추천하는 서비스를 만든다.

## 현재 단계
MVP(Minimum Viable Product) 단계. 확장성보다 명확한 구조와 유지보수성을 우선한다.

## 범위
- 대상 지역: 하남 미사
- 대상 과목: 수학
- 핵심 기능: 학원 데이터 기반 추천 (추후 AI/LLM 연동)

## 이번 단계에서 하지 않는 것
- 로그인/인증(JWT)
- OpenAI API 연동
- RAG / Vector DB / Embedding
- 관리자 페이지
- Flutter 클라이언트 구현
